Мой проект - создать сайт-портфолио для моей дочки, она ничего не соображает в компьютерах. В качестве технологии вижу использовать какой-то статик генератор сайтов, который нужно захостить на githab, позже купить доменное имя и привязать к этому сайту. Использовать для сайта красивый макет, я вышлю картинкой, спроси меня об этом, чтобы я прикрепил картинку, либу ссылку на похожий сайт.. Из базовых требований для пользователя сайта-портфолио вижу:
- создание нескольких разделов, название которых можно редактировать (к примеру, собаки, люди, ), раздел с контактной информацией, с этого можно начать. 
- Для владельца сайта - простой инструментарий по управлению контентом - наполнение сайта (к примеру, добавить новую сессию, дать ей имя и прикрепить фотографии), редактирование альбомов (удаление полно, стью альбома, удаление фотогргафий в альбоме, добавление фотографией), быстрая и удобная публикация контекта, чтобы мог ребенок прям через свой iphone управлять сайтом
Ну а дальне уточним требования в рамках нашего диалога с тобой
